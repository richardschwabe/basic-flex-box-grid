Basic Flex Box Grid

A super simple display:flex grid.

Tl;dr

Written in Sass, you can configure class names, provides 4 hard breakpoints, access to justify-content and align-items.

Introduction

The grid is based on the various other CSS frameworks. Without all the styling extras of a framework. It only focuses on a grid system, based on flexbox.

If you want to change the grid gap, column count etc, simply edit the variables.scss file. You can configure everything from there to your needs. A full list of variables and classes are at the bottom of the documentation.

The paddings, margins, etc. are calculated based on the 16px default browser font size and everything works with rem. Therefore, you could create a container with padding: 0 20rem; or similar, whatever your use case is. The .flex-x class will take 100% width at all times.

We did not include a "container" class, as we do not see a need for it. If you need a container class, you will need to do it yourself.

Basic flex-x

.flex-x is used to define a new row, which holds up to 12 columns of class .c

A column can have any width, these are defined by the numbers 1 to 12. For example, .l3 creates a 3 column wide column on viewports with at least 1200px width.

.s1-12 is used for at least 600px

.m1-12 is used for at least 1200px

.x1-12 is used for at least 1800px

Class List

For a full reference of all classes, see below the following table.

Default Class Name Applied to Description
.flex-x row Defines a new horizontal container
.flex-y row Defines a new vertical container
.c cell Defines a new cell
.s1 - .s12 cell Cell width on small and onwards
.m1 - .m12 cell Cell width on tablet portrait and onwards
.l1 - .l12 cell Cell width on tablet landscape and onwards
.x1 - .x12 cell Cell width on desktop and "onwards"
.margin row Provides horizontal margin on row and between cells. Default value is 1rem.
.padding row Provides horizontal padding on row and between cells. Default value is 1rem.
.reverse row Reverses the order of items (flex-direction)
.space-end row applies justify-content: space-end
.space-center row applies justify-content: center
.space-between row applies justify-content: space-between
.space-around row applies justify-content: space-around
.space-even row applies justify-content: space-evenly
.align-end row applies align-items: flex-end
.align-center row applies align-items: center
.align-stretch row applies align-items: stretch
.align-baseline row applies align-items: baseline
.auto cell applies flex: 1 1 0 (grow shrink basis), allows cells to fill remaining space
.s-hidden .m-hidden .l-hidden .x-hidden cell Hides item via display:none on selected screen-size only.

Sass Reference

For a full reference of all Sass variables and mixins.

Variable Default Value Description
$row-x-class flex-x Defines a new horizontal container
$row-y-class flex-y Defines a new vertical container
$cell-class c Defines a new cell
$columns 12 Defines how many columns the grid has
$column-margin-size 1rem Gap between columns
$column-padding-size 1rem Padding inside a column
$tab-port tab-port Used for tablet portrait viewport identification
$tab-port-size 600 Min width of tablet portrait screen size in px
$tab-land tab-land Used for tablet landscape viewport identification
$tab-land-size 900 Min width of tablet landscape screen size in px
$desktop desktop Used for desktop viewport identification
$desktop-size 1200 Min width of desktop screen size in px
$huge huge Used for full hd desktop viewport identification
$huge-size 1800 Min width of full hd desktop screen size in px
